Detailed Description

A simple square-wave oscillator (VCO) with pulse-width control.

Knob assignments: 1) Frequency (Hz) 2) Amplitude (0.0 – 1.0) 3) Duty Cycle (0.05 – 0.95) 4) (unused / reserved for future)
