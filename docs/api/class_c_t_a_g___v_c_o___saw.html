Detailed Description

A simple sawtooth-wave oscillator (VCO) with skew control.

Knob assignments: 1) Frequency (Hz) 2) Amplitude (0.0 – 1.0) 3) Skew (0.01 – 0.99) : position of the waveform’s peak within each cycle 4) (unused / reserved for future)
