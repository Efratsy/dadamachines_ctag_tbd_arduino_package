Detailed Description

A simple sine wave oscillator (VCO).

A simple sine wave oscillator (VCO) with optional vibrato LFO.

Knob assignments: 1) Frequency (Hz) 2) Amplitude (0.0 - 1.0) 3) Vibrato LFO Rate (Hz) 4) Vibrato LFO Depth (Hz)
