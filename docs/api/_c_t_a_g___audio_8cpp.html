Detailed Description

Implementation file for the CTAG audio library for ESP32.

Note
This implementation assumes that the I2S pin macros (e.g., PIN_I2S_MCLK, PIN_I2S_BCLK) are defined in the user's project scope, typically via a board-specific "pins_arduino.h".
